3 \mainpage A two-dimensional Biharmonic problem with the Bell triangular finite element
4 
5 In this document, we demonstrate how to solve a two-dimensional Biharmonic equation using triangular finite elements within \c
6 oomph-lib.
7 
8 First, let consider the Biharmonic equation which is mathematically expressed as follow
9 
10 \f[
11  \frac{\partial^4 u}{\partial x_1^4} + 2\frac{\partial^4 u}{\partial x_1^2\partial x_2^2} + \frac{\partial^4 u}{\partial x_2^4} = f(x_1,x_2), \ \ \ \mathbf{x} = (x_1,x_2) \in \Re^2. \ \ \ \ (1)
12 \f]
13 
14 In this study, we consider the homogeneous Biharmonic equation with the exact solution
15 \f[
16  u(x_1,x_2) = \cos(x_1)e^{x_2}.
17 \f]
18 
19 The problem will be implemented in the rectangular domain \f$ \Omega = \{(x_1,x_2) | 0 \leq x_1 \leq 2, 0\leq x_2 \leq 1 \} \f$, and we apply Dirichlet boundary conditions such that the exact solution is satisfied.
20 
21 In order to formulate the finite element implementation of the Biharmonic equation, the weak formulation must be derived.
22 The reader is referred to <A HREF="../../../intro/html/index.html">
23 another document</A> for detailed descriptions of weak formulations.
24 
25 After introducing the weight or test function \f$ \phi(x_i) \f$ and changing the differential equation (1) into the integral form, we have
26 \f[
27 0 = r = \int_{\Omega}{\left( \frac{\partial^4 u}{\partial x_1^4} + 2\frac{\partial^4 u}{\partial x_1^2\partial x_2^2} + \frac{\partial^4 u}{\partial x_2^4} - f(x_i)\right) \phi(x_i) d\Omega.} \ \ \ (2)
28 \f]
29 
30 Integrating by parts twice and using of the divergence theorem give
31 
32 \f[
33 0 = r = \int_{\Omega}{\left( \frac{\partial^2 u}{\partial x_1^2}\frac{\partial^2 \phi}{\partial x_1^2} + 2\frac{\partial^2 u}{\partial x_1^2}\frac{\partial^2 \phi}{\partial x_2^2} + \frac{\partial^2 u}{\partial x_2^2}\frac{\partial^2 \phi}{\partial x_2^2} - f(x_i)\phi(x_i)\right) d \Omega} \f]
34 \f[ + \int_{\partial \Omega} \nabla^3_n u \phi(x_i) d\partial \Omega - \int_{\partial \Omega} \nabla^2 u \frac{\partial \phi}{\partial n}(x_i) d\partial \Omega, \ \ \ \ \ (3)
35 \f]
36 
37 where \f$ \partial \Omega \f$ denotes the domain boundary and \f$ \partial/\partial n \f$ the outward normal derivative. Since the test functions satisfy homogeneous boundary
38 conditions, \f$ \left. \phi \right|_{\partial \Omega} = 0 = \left. \frac{\partial \phi}{\partial n}\right|_{\partial \Omega}, \f$ the integrals for the last two terms of equation (3) vanish.
39 
40 Therefore, an alternative version of the weak form of (1) is given by
41 
42 \f[
43 0 = r = \int_{\Omega}{\left( \frac{\partial^2 u}{\partial x_1^2}\frac{\partial^2 \phi}{\partial x_1^2} + 2\frac{\partial^2 u}{\partial x_1^2}\frac{\partial^2 \phi}{\partial x_2^2} + \frac{\partial^2 u}{\partial x_2^2}\frac{\partial^2 \phi}{\partial x_2^2} - f(x_i)\phi(x_i)\right) d \Omega.} \ \ \ \ \ (4)
44 \f]
45 
46 It is noteworthy that the Galerkin method is employed in this study so that the choice of function spaces for the solution, \f$u(x_i)\f$, and the test function, \f$\phi(x_i)\f$, is the same. Therefore, both the solution and the test function can be approximated by the same set of shape function, \f$\psi_l\f$ as
47 
48 \f[
49 u(x_i) = \sum_{l=1}^{N} U_{l}\psi_l(x_i), \ \ \ \ (5)
50 \f]
51 and,
52 \f[
53 \phi(x_i) = \sum_{k=1}^{N} \Phi_{k}\psi_k(x_i), \ \ \ \ (6)
54 \f]
55 where \f$ N \f$ is the dimension of discrete approximating to function space.
56 
57 Substituting the derivatives of (6) into (4), we have
58 
59 \f[
60 0 = r = \int_{\Omega}{\left( \frac{\partial^2 u}{\partial x_1^2}\sum_{k=1}^{N} \Phi_{k}\frac{\partial^2 \psi_{k}}{\partial x_1^2} + 2\frac{\partial^2 u}{\partial x_1^2}\sum_{k=1}^{N} \Phi_{k}\frac{\partial^2 \psi_{k}}{\partial x_2^2} + \frac{\partial^2 u}{\partial x_2^2}\sum_{k=1}^{N} \Phi_{k}\frac{\partial^2 \psi_{k}}{\partial x_2^2} - f(x_i)\sum_{k=1}^{N} \Phi_{k}\psi_{k}(x_i)\right) d \Omega,} \f]
61 
62 \f[
63 0 = r = \sum_{k=1}^{N}{\Phi_{k} r_{k}(U_1,U_2,...,U_N)}, \ \ \ \ (7)
64 \f]
65 
66 where
67 
68 \f[
69  r_{k}(U_1,U_2,...,U_N) = \int_{\Omega}{\left( \frac{\partial^2 u}{\partial x_1^2}\frac{\partial^2 \psi_{k}}{\partial x_1^2} + 2\frac{\partial^2 u}{\partial x_1^2}\frac{\partial^2 \psi_{k}}{\partial x_2^2} + \frac{\partial^2 u}{\partial x_2^2}\frac{\partial^2 \psi_{k}}{\partial x_2^2} - f(x_i)\psi_{k}(x_i)\right) d \Omega.} \ \ \ \ (8)
70 \f]
71 
72 We have that (7) must hold for any value of the coefficients \f$ \Phi_k \f$, so the coefficients \f$ U_j \f$ must satisfy the equations
73 
74 \f[
75 r_k(U_1, U_2,...,U_N) = 0, \mbox{ \ \ \ \ for } k=1,2,...,N.
76 \f]
77 
78 After substituting the derivatives of (5) into (8), the weak form of (1) can be expressed as
79 
80 \f[
81 0 = r_{k} = \int_{\Omega}\left( \sum_{l=1}^{N}{U_l\frac{\partial^2 \psi_l}{\partial x_1^2}}\frac{\partial^2\psi_k}{\partial x_1^2} + 2\sum_{l=1}^{N}{U_l\frac{\partial^2 \psi_l}{\partial x^2_1}}\frac{\partial^2\psi_k}{\partial x^2_2} + \sum_{k=1}^{N}{U_l\frac{\partial^2 \psi_l}{\partial x_2^2}}\frac{\partial^2\psi_k}{\partial x_2^2} - f(x_i)\psi_k \right) dx_1dx_2. \ \ (9)
82 \f]
83 
84 Since we solve the Biharmonic equation with the Dirichlet boundary
85 conditions in this study, the boundary terms in
86 (9) vanish. This is a consequence of every
87 test functions have to satisfy the homogeneous conditions at
88 boundaries. Now, it can be seen that the weak formulation of the equation contains the second-order derivatives of the approximation, \f$ \psi(x_i) \f$ of the solution. Hence, in order to make the integral exists, the approximation and its first-order derivative are required to be continuous. Therefore, the \f$ C^1 \f$-interpolations are required.
89 
90 Hence, the \c BellElement is introduced in this document to provide
91 the continuously differentiable shape functions to interpolate the
92 solution of a Biharmonic equation. This kind of element arises in order to satisfy \f$ C^1 \f$ continuity of the solutions of a fourth-order problem.
93 
94 Specifically, we shall provide
95 
96 - a general descriptions of the Bell triangular finite element
97 .
98 - numerical results of solving the Biharmonic equation with the \c BellElement
99 .
100 and
101 - how to implement the \c BellElement in \c oomph-lib using the
102 Biharmonic equation as a case study.
103 .

108 \section overview Overview of the BellElement
109 
110 The constructed \c BellElement provides a discretisation of the variational principle (9) with two-dimensional, three-node triangular finite elements. In this element, the subparametric idea is employed. This means that the degree of geometric shape functions (shape function approximated a geometry) is defined to be lower than the degree of polynomials for parametric shape functions (shape function approximated a variable). In this study, a geometry is approximated by the linear Lagrange polynomials while the Bell shape functions are employed to interpolate variables.
111 
112 The two-dimensional linear Lagrange shape functions defined over a triangle are parametrised by the local coordinates \f$ s_1,s_2 \in [0,1]\f$ and are defined as
113 
114 \f[ \psi_{1}(s_1,s_2) = s_1, \ \ \ \psi_{2}(s_1,s_2) = s_2, \ \ \ \psi_{3}(s_1,s_2) = 1-s_1-s_2. \ \ \ \ \ (10)\f]
115 
116 Therefore, a position \f$ {\bf x} = (x_1,x_2) \in \Re^2 \f$ is approximated by
117 
118 \f[
119 x_i(s_1,s_2) = \sum_{j=1}^{3} X_{ij} \ \psi_{j}(s_1,s_2)
120 \mbox{\ \ \ for $i=1,2$},
121 \ \ \ \ \ \ \ \ (11)
122 \f]
123 where \f$ X_{ij} \f$ are the nodal positions in the direction \f$ i^{th} \f$ at node \f$ j \f$.
124 
125 Similar to a geometric approximation, a variable can be approximated as
126 
127 \f[
128 u_i(x_1,x_2) = \sum_{l=1}^{3} \sum_{k=1}^{6}U_{ilk} \ \varphi_{lk}(\lambda_1,\lambda_2,\lambda_3),
129 \f]
130 where \f$ x_1,x_2 \in \Omega \f$ are the element's 2D global coordinates. The function \f$ \varphi_{lk}\f$ are the two-dimensional Bell shape functions parametrised by area coordinates \f$ \lambda_1, \lambda_2, \lambda_3\f$ which we will describe in the next two subsections.

134 \subsection area_coor Area coordinates
135 
136 The area coordinate system or Barycentric coordinates are defined on a triangle which is expressed in terms of its three vertices. We consider a triangle defined by three vertices \f$ {\bf x}_{1}, {\bf x}_{2}, {\bf x}_{3}\f$ as
137 
138 \image html triangle.gif "The geometry of a triangle which has three nodes. "
139 \image latex triangle.eps "The geometry of a triangle which has three nodes. " width=0.55\textwidth
140 
141 The barycentric coordinates of point \f$ {\bf p} = (x,y)\f$ in the triangle
142 are simply defined to be ratios of triangular areas as,
143 \f[ \lambda_1 = \frac{A_1}{A}, \ \ \ \lambda_2 = \frac{A_2}{A}, \ \ \ \lambda_3 = \frac{A_3}{A}. \ \ \ \ (12)
144 \f]
145 
146 Area \f$ A \f$ is the total area of the triangle and can be computed
147 as
148 \f[ A = \frac{1}{2}\left(\left(x_2y_3-x_3y_2\right) + \left(x_3y_1-x_1y_3\right) + \left(x_1y_2-x_2y_1\right)\right).\f]
149 
150 Also, \f$ A_j, j=1,2,3 \f$ are the sub-area of the triangle described
151 in the figure. They can be computed as
152 \f[ A_1 = \frac{1}{2}\left(\left(x_2y_3-x_3y_2\right) +
153 \left(x_3y-xy_3\right) + \left(xy_2-x_2y\right)\right),\f]
154 \f[ A_2 = \frac{1}{2}\left(\left(xy_3-x_3y\right) +
155 \left(x_3y_1-x_1y_3\right) + \left(x_1y-xy_1\right)\right),\f]
156 \f[ A_3 = \frac{1}{2}\left(\left(x_2y-xy_2\right) + \left(xy_1-x_1y\right) + \left(x_1y_2-x_2y_1\right)\right),\f]
157 
158 where \f$ x_i,y_i \f$ denotes the \f$ x- \f$ and \f$ y- \f$ coordinates of the point \f$ \mathbf{x}_i;i =1,2,3. \f$ These ratios of areas form dimensionless coordinates in the plane defined by points \f$ {\bf x}_{1}, {\bf x}_{2}, {\bf x}_{3}. \f$
159 
160 A barycentric combination of three points takes the form: \f$ {\bf p}
161 = \lambda_1{\bf x}_{1} + \lambda_2{\bf x}_{2} + \lambda_3{\bf
162 x}_{3}\f$ where \f$ \lambda_1+\lambda_2+\lambda_3 = 1.\f$ The third area coordinate \f$ \lambda_3 \f$ can be
163 expressed as \f$ \lambda_3 = 1-\lambda_1-\lambda_2. \f$
164 
165 The three vertices have the barycentric coordinates as
166 \f[ {\bf x}_{1} = \left(1,0,0\right), \ \ \ {\bf x}_{2} = \left(0,1,0\right), \ \ \ {\bf x}_{3} = \left(0,0,1\right). \f]

171 \subsection bell_shape The Bell shape functions
172 
173 The shape functions of the Bell element are defined over a triangle
174 and derived from the quintic polynomials with 18 degrees of
175 freedom. The element comprises three nodes with six degrees of
176 freedom at each corner node including the value, and all first and second derivatives; \f$ u, u_{x}, u_{y}, u_{xx}, u_{yy}, u_{xy}. \f$ The graphical description of the Bell element is illustrated below.
177 
178 \image html bell_element.gif "(Left)The Bell element with 18 degrees of freedom; 6 dofs at vertices. (Right) The description of all parameters utilised within the derivation of the Bell shape functions. "
179 \image latex bell_element.eps "(Left)The Bell element with 18 degrees of freedom; 6 dofs at vertices. (Right) The description of all parameters utilised within the derivation of the Bell shape functions. " width=0.75\textwidth
180 
181 The shape functions of the Bell element are parametrised by area
182 coordinates, \f$\lambda_j\f$. The subscripts \f$ik\f$ of the shape
183 functions \f$\psi_{ik}\f$ mean that they are evaluated at node \f$i\f$
184 with \f$k^{th}\f$ degree of freedom where \f$k=1\f$ denotes degree of
185 freedom that corresponds to the unknown value, \f$k=2,3\f$ correspond to the first derivatives with respect to the first and second global coordinates, \f$k=4,5\f$ correspond to the second derivatives with respect to the first and second global coordinates, and \f$k=6\f$ corresponds to the mixed derivative.
186 
187 Unlike the Lagrangian interpolation functions, the Bell shape functions are derived by interpolating nodal derivatives as well as nodal displacements. Also, they have to satisfy the properties that
188 \f$\psi_{i1} = 1\f$ at node \f$i\f$ and \f$\psi_{i1} = 0\f$ at other
189 nodes. Furthermore, all of the first and the second derivatives of
190 \f$\psi_{i1} = 0\f$ at all nodes. Also, \f$\partial_{x}\psi_{i2} =
191 1\f$ at node \f$i\f$ and \f$\partial_{x}\psi_{i2} = 0\f$ at other
192 nodes. Similarly for the first derivative with respect to \f$y\f$, we
193 have that \f$\partial_{y}\psi_{i3} = 1\f$ at node \f$i\f$ and
194 \f$\partial_{y}\psi_{i3} = 0\f$ at other nodes. These shape functions
195 associated with the first derivatives are zero at all nodes for \f$\psi_{im}\f$, \f$\partial_{xx}\psi_{im}\f$, \f$\partial_{yy}\psi_{im}\f$, \f$\partial_{xy}\psi_{im}, \forall m=2,3\f$. Moreover, \f$\partial_{xx}\psi_{i4}(\mathbf{x}_{k}) = \delta_{ik}\f$, \f$\partial_{yy}\psi_{i5}(\mathbf{x}_{k}) = \delta_{ik}\f$, and \f$\partial_{xy}\psi_{i6}(\mathbf{x}_{k}) = \delta_{ik}\f$. Furthermore, these shape functions
196 associated with the second derivatives are zero at all nodes for \f$\psi_{im}\f$, \f$\partial_{x}\psi_{im}\f$, and \f$\partial_{y}\psi_{im}, \forall m=4,5,6\f$.
197 
198 The Bell shape functions are mathematically expressed as
199 
200 \f[ \psi_{11}(\lambda_1,\lambda_2,\lambda_3) = \lambda_1^5 + 5\lambda_1^4\left( \lambda_2+\lambda_3\right) + 10\lambda_1^3\left( \lambda_2+\lambda_3\right)^2 + 30\lambda_1^2\lambda_2\lambda_3\left( l_3\lambda_2 + l^{\prime}_2\lambda_3\right), \f]
201 \f[ \psi_{12}(\lambda_1,\lambda_2,\lambda_3) = 3b_1\lambda_1^2\lambda_2\lambda_3\left( \lambda_2-\lambda_3\right) +
202 \lambda_1^3\left(b_3\lambda_2 -b_2\lambda_3 \right)\left( \lambda_1+4\lambda_2+4\lambda_3\right) \f]
203 \f[ + 15\lambda_1^2\lambda_2\lambda_3\left( b_3l_3\lambda_2-b_2l^{\prime}_2\lambda_3\right), \f]
204 \f[ \psi_{13}(\lambda_1,\lambda_2,\lambda_3) = -3c_1\lambda_1^2\lambda_2\lambda_3\left( \lambda_2-\lambda_3\right) -
205 \lambda_1^3\left(c_3\lambda_2 -c_2\lambda_3 \right)\left( \lambda_1+4\lambda_2+4\lambda_3\right) \f]
206 \f[ - 15\lambda_1^2\lambda_2\lambda_3\left( c_3l_3\lambda_2-c_2l^{\prime}_2\lambda_3\right), \f]
207 \f[ \psi_{14}(\lambda_1,\lambda_2,\lambda_3) = 1/2\lambda_1^3\left(
208 b_3^2\lambda_2^2+b_2^2\lambda_3^2\right) + \lambda_1^2\lambda_2\lambda_3\left(-b_2b_3\lambda_1 + b_3b_1\lambda_2
209 + b_1b_2\lambda_3 \right) \f]
210 \f[ + 5/2\lambda_1^2\lambda_2\lambda_3\left( b_3^2l_3\lambda_2+b_2^2l^{\prime}_2\lambda_3\right), \f]
211 \f[ \psi_{15}(\lambda_1,\lambda_2,\lambda_3) = 1/2\lambda_1^3\left(
212 c_3^2\lambda_2^2+c_2^2\lambda_3^2\right) + \lambda_1^2\lambda_2\lambda_3\left(-c_2c_3\lambda_1 + c_3c_1\lambda_2
213 + c_1c_2\lambda_3 \right) \f]
214 \f[ + 5/2\lambda_1^2\lambda_2\lambda_3\left( c_3^2l_3\lambda_2+c_2^2l^{\prime}_2\lambda_3\right), \f]
215 \f[ \psi_{16}(\lambda_1,\lambda_2,\lambda_3) = b_1c_1\lambda_1^2\lambda_2\lambda_3\left(
216 \lambda_1+\lambda_2+\lambda_3\right) + b_2c_2\lambda_1^2\lambda_3\left( \lambda_2\lambda_3 - \lambda_3\lambda_1 - \lambda_1\lambda_2 -
217 \lambda_2^2\right) \f]
218 \f[ + b_3c_3\lambda_1^2\lambda_2\left( \lambda_2\lambda_3 - \lambda_3\lambda_1 - \lambda_1\lambda_2 - \lambda_3^2\right) - 5\lambda_1^2\lambda_2\lambda_3\left( b_2c_2l^{\prime}_2\lambda_3 + b_3c_3l_3\lambda_2\right). \ \ \ \ (13)\f]
219 
220 These are shape functions associated with the first node with its 6 degrees of freedom. The shape functions at another two nodes can be obtained by performing a cyclic permutation of the Barycentric coordinates in (13). The parameters \f$ b_i, c_i \forall i = 1,2,3, \f$ appearing in the equations can be obtained from
221 
222 \f[ b_1 = x_3 - x_2, \ \ \ b_2 = x_1-x_3, \ \ \ b_3 = x_2 - x_1, \f]
223 \f[ c_1 = y_2 - y_3, \ \ \ c_2 = y_3-y_1, \ \ \ c_3 = y_1 - y_2, \f]
224 where \f$x_i,y_i; i=1,2,3,\f$ are components of a vertex
225 \f$\mathbf{x}_i. \f$
226 
227 Note that all the first and second derivatives can be obtained by the help of the chain rule as
228 
229 \f[
230 \frac{\partial \psi_{ij}}{\partial x_{\alpha}} = \frac{\partial \psi_{ij}}{\partial \lambda_{k}}\frac{\partial \lambda_{k}}{\partial x_{\alpha}}, \f]
231 \f[ \frac{\partial^2 \psi_{ij}}{\partial x_{\alpha}\partial x_{\beta}} = \frac{\partial^2 \psi_{ij}}{\partial \lambda_{m}\partial \lambda_{n}}\frac{\partial \lambda_{m}}{\partial x_{\alpha}}\frac{\partial \lambda_{n}}{\partial x_{\beta}} + \frac{\partial^2 \lambda_{k}}{\partial x_{\alpha}\partial x_{\beta}}\frac{\partial \psi_{ij}}{\partial \lambda_{k}}, \f]
232 
233 and the derivatives of area coordinates with respect to global coordinates can be obtained from considering (12) with the substitutions of \f$ b_i, c_i; i=1,2,3. \f$ They are expressed as follows:
234 
235 \f[ \frac{\partial \lambda_{1}}{\partial x} = \frac{c_1}{2A}, \ \ \ \frac{\partial \lambda_{2}}{\partial x} = \frac{c_2}{2A}, \ \ \ \frac{\partial \lambda_{3}}{\partial x} = \frac{c_3}{2A}, \f]
236 
237 \f[ \frac{\partial \lambda_{1}}{\partial y} = \frac{b_1}{2A}, \ \ \ \frac{\partial \lambda_{2}}{\partial y} = \frac{b_2}{2A}, \ \ \ \frac{\partial \lambda_{3}}{\partial y} = \frac{b_3}{2A}. \f]
238 
239 Here are graphical Bell shape functions at the first node, \f$
240 \psi_{1j}, j=1,...,6,\f$ evaluated at \f$ {\bf x}_1 \f$. The figures show that the Bell shape function at the first node have to satisfy the properties that
241 \f$\psi_{11} = 1\f$ at node \f$1\f$ and \f$\psi_{11} = 0\f$ at other
242 nodes. Furthermore, all of the Bell shape functions corresponded to all first and second derivatives \f$ \psi_{1j} = 0, j=2,...,6 \f$ at all nodes.
243 
244 \image html shapes.gif "The Bell shape functions that correspond to the 6 degrees of freedom defined previously at the first node. Here the shape functions are evaluated at x1 which show the appreciation of the crucial properties in the Bell shape functions. "
245 \image latex shapes.eps "The Bell shape functions that correspond to the 6 degrees of freedom defined previously at the first node. Here the shape functions are evaluated at x1 which show the appreciation of the crucial properties in the Bell shape functions. " width=1.05\textwidth

247 Note that the Bell shape functions and also the first and second
248 derivatives are differentiated with respect to global coordinates. Therefore, the Jacobian of mapping between global to local coordinates is no longer needed to derive the derivatives of the shape functions for this type of element.

312 \section reslt Results
313 
314 Before the numerical results obtained from solving the 2D Biharmonic equation with the Bell finite element will be illustrated, the boundary conditions have to be specified. Since the Bell finite element is employed, the boundary specifications have to correspond with degrees of freedom defined for the \c BellElement in the previous section.
315 
316 Since we consider the Biharmonic equation with the Dirichlet boundary conditions, we have that the physical conditions that allow to be pinned on boundaries are the value and the normal derivatives. However, there is no normal derivative defined as a degree of freedom to be specified for the Bell shape function. Therefore, the first-order derivatives have to be imposed instead and the boundary specification can be worked out from the normal derivative on that boundary.
317 
318 Furthermore, the values of the unknown and the first-order derivatives that have to be specified on the boundaries of the domain for the Bell element can be obtained from
319 
320 \f[ u(x_1,x_2) = \cos(x_1)e^{x_2} , \ \ \ u_{x_1}(x_1,x_2) =
321 -\sin(x_1)e^{x_2}, \ \ \ u_{x_2}(x_1,x_2) = \cos(x_1)e^{x_2}. \ \ \
322 (14) \f]
323 
324 For the rest of degrees of freedom associated with the
325 second-order derivatives, they are derived conditions that can
326 be taken care by the natural boundary conditions. In this study, we
327 set them to be pinned in order to reduce the number of degrees of
328 freedom in the problem. The values of the second-order derivatives
329 that have to be specified on the boundaries of the domain for the Bell
330 element can be worked out from (14) and are expressed as
331 
332 \f[ u_{x_1x_1}(x_1,x_2) = -\cos(x_1)e^{x_2} , \ \ \ u_{x_1x_2}(x_1,x_2) = -\sin(x_1)e^{x_2}, \ \ \ u_{x_2x_2}(x_1,x_2) = \cos(x_1)e^{x_2}. \f]
