Detailed Description

template<unsigned DIM, unsigned NNODE_1D>
class oomph::BellBiharmonicElement< DIM, NNODE_1D >

BellBiharmonicElement elements are a subparametric scheme with linear Lagrange interpolation for approximating the geometry and the C1-functions for approximating variables.
