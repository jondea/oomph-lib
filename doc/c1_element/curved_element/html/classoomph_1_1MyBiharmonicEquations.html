Detailed Description

template<unsigned DIM, unsigned NNODE_1D>
class oomph::MyBiharmonicEquations< DIM, NNODE_1D >

A class for all subparametric elements that solve the 2D- Biharmonic equations.

This contains the generic maths. Shape functions, geometric mapping etc. must get implemented in derived class.
