class oomph\+::\+Advection\+Diffusion\+Equations$<$ D\+I\+M $>$}

A class for all elements that solve the Advection Diffusion equations using isoparametric elements. 

\[ \frac{\partial^2 u}{\partial x_i^2} = Pe w_i(x_k) \frac{\partial u}{\partial x_i} + f(x_j) \] This contains the generic maths. \mbox{\hyperlink{classoomph_1_1Shape}{Shape}} functions, geometric mapping etc. must get implemented in derived class.
