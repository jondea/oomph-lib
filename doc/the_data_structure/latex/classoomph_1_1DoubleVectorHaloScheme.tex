\doxysubsection{Detailed Description}
A class that stores the halo/haloed entries required when using a \mbox{\hyperlink{classoomph_1_1DoubleVectorWithHaloEntries}{Double\+Vector\+With\+Halo\+Entries}}. This is a separate class so thay many different Vectors can share the same object. The constructor requires the distribution of the \mbox{\hyperlink{classoomph_1_1DoubleVector}{Double\+Vector}} (if you pass in a different distribution things will go badly wrong) and a vector that specifies which G\+L\+O\+B\+AL eqn numbers are required on each processor.
