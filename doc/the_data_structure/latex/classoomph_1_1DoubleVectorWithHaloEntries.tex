\doxysubsection{Detailed Description}
===================================================================== An extension of \mbox{\hyperlink{classoomph_1_1DoubleVector}{Double\+Vector}} that allows access to certain global entries that are not stored locally. Synchronisation of these values must be performed manually by calling the \mbox{\hyperlink{classoomph_1_1DoubleVectorWithHaloEntries_a6d5b6e2c1c80d4fb468c851030d62492}{synchronise()}} function. Synchronisation can only be from the haloed to the halo, but the local halo entries can all be summed and stored in the \hypertarget{classoomph_1_1DoubleVectorWithHaloEntries_autotoc_md3}{}\doxysubsection{haloed value.}\label{classoomph_1_1DoubleVectorWithHaloEntries_autotoc_md3}
