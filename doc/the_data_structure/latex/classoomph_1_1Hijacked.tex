\mbox{\hyperlink{classoomph_1_1Hijacked}{Hijacked}} elements are elements in which one or more \mbox{\hyperlink{classoomph_1_1Data}{Data}} values that affect the element\textquotesingle{}s residuals, are determined by another element -- the data values are then said to have been hijacked by another element. The main functionality added by the \mbox{\hyperlink{classoomph_1_1Hijacked}{Hijacked}} element class is that it wipes out those entries in the element\textquotesingle{}s residual vector and those rows in the element\textquotesingle{}s Jacobian matrix that are determined by the \char`\"{}other\char`\"{} elements that have hijacked the values. Note that for continuation in homotopy parameters, it may be desriable to multiply the residuals and corresponding jacobian entries by a \char`\"{}homotopy parameter\char`\"{}. The value of this parameter can be set by assigning \mbox{\hyperlink{classoomph_1_1HijackedElementBase_affdb1888fb6b2de051bc0345e6f5d934}{residual\+\_\+multiplier\+\_\+pt()}} which has a default value of zero. Note\+: it would be possible to extend the functionality so that different residuals are multiplied by different values, but will this ever be required?
