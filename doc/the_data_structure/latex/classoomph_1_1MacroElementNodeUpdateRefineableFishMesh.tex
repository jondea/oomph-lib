class oomph\+::\+Macro\+Element\+Node\+Update\+Refineable\+Fish\+Mesh$<$ E\+L\+E\+M\+E\+N\+T $>$}

Refineable fish shaped mesh with Macro\+Element-\/based node update. The fish\textquotesingle{}s back is represented by a specified geometric object. Some or all of the geometric \mbox{\hyperlink{classoomph_1_1Data}{Data}} in that geometric object may contain unknowns in the global \mbox{\hyperlink{classoomph_1_1Problem}{Problem}}. The dependency on these unknowns is taken into account when setting up the Jacobian matrix of the elements. For this purpose, the element (whose type is specified by the template parameter) must inherit from \mbox{\hyperlink{classoomph_1_1MacroElementNodeUpdateElementBase}{Macro\+Element\+Node\+Update\+Element\+Base}}. ~\newline
