class oomph\+::\+Navier\+Stokes\+Surface\+Drag\+Torque\+Element$<$ E\+L\+E\+M\+E\+N\+T $>$}

A class of elements that allow the determination of the drag and toque, relative to a given centre of rotation, along a domain boundary. The element operates as a \mbox{\hyperlink{classoomph_1_1FaceElement}{Face\+Element}} and attaches itself to a bulk element of the type specified by the template argument.
