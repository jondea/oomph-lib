An interface to the Trilinos Aztec\+OO classes allowing it to be used as an Oomph-\/lib \mbox{\hyperlink{classoomph_1_1LinearSolver}{Linear\+Solver}}. The Aztec\+OO solver is a Krylov Subspace solver; the solver type (either \mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{G\+M\+R\+ES}} or \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+C\+G\+Stab}}) can be set using \mbox{\hyperlink{classoomph_1_1TrilinosAztecOOSolver_a9169831da7460eec52071e8b3938bf97}{solver\+\_\+type()}}. This solver can be preconditioned with Trilinos Preconditioners (derived from \mbox{\hyperlink{classoomph_1_1TrilinosPreconditionerBase}{Trilinos\+Preconditioner\+Base}}) or Oomph-\/lib preconditioners (derived from \mbox{\hyperlink{classoomph_1_1Preconditioner}{Preconditioner}}). Preconditioners are set using \mbox{\hyperlink{classoomph_1_1IterativeLinearSolver_a800f699ec8dda290105807cb91297850}{preconditioner\+\_\+pt()}}.
